# EasyTools

Публичный репозиторий содержит только пользовательскую бинарную поставку EasyTools.

Страница проекта: `https://asppel.github.io/easytools/`

## Скачать

- `downloads/EasyTools.Setup.exe` - установщик EasyTools для AutoCAD 2025
- `downloads/EasyTools.Setup.exe.sha256` - контрольная сумма SHA-256

## Установка

1. Скачайте `EasyTools.Setup.exe`.
2. Запустите установщик.
3. Нажмите `Установить`.
4. Установщик развернет `EasyTools.bundle` в `%APPDATA%\Autodesk\ApplicationPlugins`.

## Состав поставки

- Только скомпилированные бинарные файлы.
- Исходный код в публичный репозиторий не входит.

Версия поставки: 1.0.0.0
