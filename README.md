# EasyTools

EasyTools — модульная платформа для инженерной работы в AutoCAD. Проект развивается как единая система, где платформа, host, runtime, интерфейс и прикладные наборы работают в общей архитектуре и не размывают границы ответственности.

## Что такое EasyTools

EasyTools помогает выстраивать прикладные инженерные инструменты на общей платформенной основе:

- единый рабочий контур внутри AutoCAD;
- наборы как предметные модули со своей логикой, данными и интерфейсами;
- расширяемая архитектура без хаотичных надстроек;
- аккуратная работа с карточками данных, действиями, массовыми сценариями и экспортом.

## Основная функциональность платформы

Публичная линия EasyTools ориентирована на надёжную платформу для дальнейшего профессионального роста:

- загрузка и исполнение модульных наборов;
- единый UI-контур для выбора набора, работы с карточками и настройками;
- хранение и обработка данных ET-объектов;
- интеграция действий наборов в общий runtime AutoCAD;
- подготовленный release- и deploy-контур для публичной поставки.

## Как устроены наборы

В EasyTools набор — это самостоятельная предметная зона внутри общей платформы. Набор может содержать:

- модель данных;
- редактор карточки;
- массовое редактирование;
- экспортные сценарии;
- собственные ресурсы и прикладную логику.

Такой подход позволяет развивать платформу и наборы согласованно, не ломая архитектурные границы.

## Что входит в публичную поставку

Публичная поставка EasyTools сосредоточена на платформе:

- основной runtime и UI-контур EasyTools для AutoCAD;
- публичный установщик;
- публичная release-витрина и история релизов.

Прикладные наборы могут развиваться и публиковаться отдельно от базовой платформы. Отдельные наборы и специализированные функции предоставляются индивидуально по запросу.
