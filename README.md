# EasyTools

EasyTools — дополнение для AutoCAD с отдельным интерфейсом и индивидуальными наборами инструментов, которые заменяют и оптимизируют разработку проектной документации.

## Назначение

EasyTools помогает ускорять прикладную работу в AutoCAD, упрощает выполнение узконаправленных задач и делает пользовательский рабочий процесс более удобным и предсказуемым.

## Состав

- дополнение для AutoCAD;
- отдельный интерфейс;
- индивидуальные пользовательские наборы инструментов по запросу.

## Требования

- AutoCAD 2025;
- Windows 10 или Windows 11.

## Достоинства

- простой и интуитивный интерфейс;
- выполнение определённых узконаправленных задач;
- ускорение пользовательской работы в проектировании.
