Публичная поставка EasyTools

EasyTools 1.0.0

Готовая пользовательская сборка для AutoCAD 2025. Один установщик, одна точка входа, одна рабочая среда для наборов, карточек данных, переноса параметров и выпускных результатов.

Суть поставки

Рабочий путь: набор → карточка данных → перенос → регистрация.

Репозиторий содержит только бинарные материалы, установщик и документы релиза. Исходный код в публичную поставку не входит.

Загрузка

Используйте версионированный файловый релиз 1.0.0. В нём размещён установщик, контрольная сумма и краткое описание поставки без дублирования лишних материалов.

Скачать EasyTools.Setup.exe Открыть описание релиза Открыть репозиторий

Формат

Только бинарная поставка

Публичная часть проекта предназначена для распространения готовой версии среди пользователей.

Версия

1.0.0

Первая оформленная пользовательская поставка EasyTools для публичной загрузки.

Совместимость

AutoCAD 2025

Установка выполняется в профиль пользователя без ручной сборки и без подготовки исходников.

Состав версии 1.0.0

  • release/1.0.0/EasyTools.Setup.exe — установщик поставки.
  • release/1.0.0/EasyTools.Setup.exe.sha256 — контрольная сумма.
  • release/1.0.0/RELEASE.md — описание версии.
  • %APPDATA%\Autodesk\ApplicationPlugins\EasyTools.bundle — целевой путь установки.

Установка

  • Откройте каталог релиза 1.0.0.
  • Скачайте и запустите EasyTools.Setup.exe.
  • При необходимости сверьте SHA-256 по файлу EasyTools.Setup.exe.sha256.
  • Нажмите кнопку установки в окне EasyTools Setup.
  • Установщик полностью обновит EasyTools.bundle в профиле AutoCAD.
