EasyTools

Публичная поставка для пользовательской установки. Репозиторий содержит только скомпилированные файлы и готовый инсталлятор для AutoCAD 2025.

Загрузка

Скачайте единый установщик и разверните актуальный EasyTools.bundle в пользовательский профиль AutoCAD.

Скачать EasyTools.Setup.exe Открыть репозиторий

Установка

  1. Скачайте EasyTools.Setup.exe.
  2. Запустите установщик.
  3. Нажмите кнопку установки в окне EasyTools Setup.
  4. Пакет будет развернут в %APPDATA%\Autodesk\ApplicationPlugins\EasyTools.bundle.
