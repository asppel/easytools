Кратко

Версия 1.0.0 — первая оформленная публичная поставка EasyTools для AutoCAD 2025. Поставка предназначена как базовая свободно доступная платформа EasyTools.

Состав поставки

  • EasyTools — установочный файл публичной версии.
  • SHA-256 — контрольная сумма установочного файла.

Установка

  • Скачайте файл EasyTools.
  • При необходимости сверьте SHA-256.
  • Запустите установочный файл.
  • Подтвердите установку в окне EasyTools.
