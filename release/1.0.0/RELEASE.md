# EasyTools 1.0.0

## Краткая суть проекта

EasyTools — это рабочая среда для AutoCAD, которая объединяет работу с наборами, объектами и выпускными данными в одном пользовательском интерфейсе.

## Назначение версии

Версия `1.0.0` открывает публичную поставку EasyTools как готового пользовательского продукта. Это первая оформленная сборка, которую можно скачать, установить и использовать как базовую точку входа в рабочий процесс EasyTools.

## Что даёт EasyTools

- единое пространство работы внутри AutoCAD
- структурированную работу с параметрами объектов
- понятный пользовательский сценарий без лишних промежуточных действий
- основу для выпуска инженерных данных в одном процессе

## Что входит в релиз

- `EasyTools.Setup.exe` — установщик EasyTools
- `EasyTools.Setup.exe.sha256` — контрольная сумма SHA-256
