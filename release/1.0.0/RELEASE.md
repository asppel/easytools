# Релиз EasyTools 1.0.0

Версия `1.0.0` — публичная пользовательская поставка базовой платформы EasyTools.

## Состав релиза

- `EasyTools.Setup.exe` — установщик базовой платформы версии `1.0.0`
- `EasyTools.Setup.exe.sha256` — контрольная сумма SHA-256

## Назначение версии

- Установка готового `EasyTools.bundle` в профиль пользователя AutoCAD
- Поставка без исходного кода и без прикладных наборов
- Единая точка входа для пользовательского внедрения

## Прикладные наборы

- Прикладные наборы не входят в публичный релиз
- Наборы поставляются отдельно по запросу

## Целевой путь установки

- `%APPDATA%\Autodesk\ApplicationPlugins\EasyTools.bundle`

## Совместимость

- AutoCAD 2025
