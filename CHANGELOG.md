# Changelog

Все заметные изменения EasyTools фиксируются в этом файле.

## [1.0.1] - 2026-03-29

Второй публичный выпуск платформы EasyTools.

### Included

- обновлённый shell и очищенные пользовательские строки;
- расширенная поддержка наборов;
- актуализированный release-контур платформы без `Contents\Sets`;
- компактный installer без зависимости от `.NET 8`.

### Release Notes

- публичная поставка по-прежнему содержит только платформу;
- installer переведён на `.NET Framework 4.8`, что существенно уменьшает размер файла;
- прикладные наборы в публичный installer не входят.

## [1.0.0] - 2026-03-26

Первый публичный выпуск платформы EasyTools.

### Included

- базовая платформа EasyTools для AutoCAD;
- public binary-only installer;
- release-модель без включения прикладных наборов в платформенный installer.
